# Murmuro — Prototype

Sleek dark-interface prototype for **Murmuro.ai**, an enterprise operational intelligence
tool. Built with React, Tailwind CSS, Lucide React, and Framer Motion.

## Run

```bash
npm install
npm run dev
```

Then open http://localhost:5173.

## Views

- **Landing** — hero, gradient headline, glass preview window.
- **Executive Overview** — metrics grid + priority inquiries (default dashboard view).
- **Inquiry Detail** — vertical operational map with bottleneck callout, recommendation card, and employee voice quotes. Click any inquiry row in the overview or sidebar to open it.

## Stack

- React 18 + Vite
- Tailwind CSS 3
- Framer Motion (page + element transitions)
- Lucide React (icons)
- Google Fonts: Space Grotesk (display), Inter (body)
